# Source Provenance

This document summarizes the principal material classes represented in the public repository and their role in the released package.

## Repository Components

- `src/` contains the implementation used for detection, segmentation-orientation, OCR, model export, and runtime support.
- `notebooks/` contains companion notebooks retained as workflow references for the major experimental components.
- `metadata/training_tables/` contains release-ready tabular metadata preserved for inspection of training inputs and configuration support.
- `evidence/` contains compact validation tables, supporting figures, and selected training-log artifacts aligned with manuscript review needs.
- `scripts/manuscript_figures/` contains retained figure-generation scripts associated with the manuscript presentation layer.
- `scripts/` also includes repository-preparation utilities retained for transparency of the released package structure.

## Availability Note

The repository preserves the technical materials required for code inspection, methodological tracing, and manuscript support in a publication-oriented format. Requests for additional material should be addressed directly to the corresponding author.
