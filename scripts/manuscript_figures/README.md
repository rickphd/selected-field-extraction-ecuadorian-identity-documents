# Manuscript Figure Scripts

This directory contains figure-generation scripts retained to support the visual materials associated with the manuscript.

- `generate_fig6_yolo_validation_metrics.py` regenerates the supplementary YOLO validation figure from `evidence/yolo_validation/results.csv`.
- `generate_fig8_ocr_val_cer_curves.py` regenerates the supplementary OCR validation CER figure from `evidence/supplementary_validation/ocr_tensorboard_scalars.csv`.
- The remaining scripts preserve manuscript figure-generation utilities associated with the released study materials.

Requests for additional figure assets or manuscript-related materials should be addressed to the corresponding author listed in the paper.
